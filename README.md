# Student Instructions for GitHub Classroom Assignment: OOP Part 2

## 1. Clone the Repository to Your Computer

Copy the repository URL from GitHub (look for the green Code button).
Open your terminal/command prompt and run:

```
cd <path where folder should be created>
git clone <repository-URL>
cd <repository-folder>
npm init -y
```

## 2. Open and Edit the Code

- Open the project folder in your code editor (like VS Code).
- Read the instructions in the README.md.
- Read tasks in TASK.md.
- Write your solution inside the created files. You will need to create a new file LibraryItem.js for this part.
- To run your function in terminal use a command:

```
  node index.js
```

## 3. Commit and Push Your Changes

- Save your changes (`Ctrl+S` in file).
- Run these commands in your terminal:
  ```
  git add .
  git commit -m "Completed the assignment Part 2"
  git push origin main
  ```
  This uploads your solution back to your GitHub repository.

## 4. Submit Your Assignment

Click a "Submit" button on GitHub Classroom or notify them once done.

## 5. Check Feedback

Your teacher review your code and give feedback via GitHub.

## 6. Helpful Tips

> [!TIP]
>
> - If you get stuck, check your code carefully for syntax errors.
> - Use the function name exactly as in the task description
> - Pay attention to new.target when creating abstract classes.
> - In each task add function execution and module.export as in task1.js before code and test running
> - Use console.log() to debug your code.
> - Commit your changes often — it helps track your progress.
> - Ask your teacher if you’re stuck, unsure about the instructions or how to submit.
